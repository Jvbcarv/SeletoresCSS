O que é um seletor?


Um seletor CSS é a primeira parte de uma regra CSS. É um padrão de elementos e outros termos que informam ao navegador quais elementos HTML devem ser selecionados para que os valores de propriedade CSS dentro da regra sejam aplicados a eles.

Para que serve o asterisco em CSS?


* é chamado de seletor universal. Esse seletor representa todos os elementos serão afetados pelas definições de estilo ali colocadas. É claro que deve-se ter cuidado ao usar o asterisco, para não afetar configurações padrões de alguns elementos.

O que é uma classe em CSS?


Em CSS, uma classe é um grupo de elementos iguais ou semelhantes. Você pode ter quantos elementos quiser em uma classe. E cada elemento pode ser membro de várias classes. Cada classe possui atributos CSS (como cor e tamanho da fonte) que são específicos dessa classe.
